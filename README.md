# Comparative-Sales-Dashboard
Power BI dashboard with time-intelligence measures and 90-day forecast

## Project Overview
Analysed two business units, cleaning separate datasets to build parallel Power BI dashboards.

## Key Features
- Time-intelligence measures
- 90-day forecast model

## Key Insight
One unit was a declining seasonal operation; the other was a 7x larger growth engine.

## Tools Used
- Power BI
- DAX
